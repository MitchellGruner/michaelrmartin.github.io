QORKD v1

Project Information

  • Project date: September 2022
  • Project URL: QORKD v1

A wine recommendation app for the casual Epicure. QORKD allows users to post and share their favorite wines with friends, make recommendations to each other, and receive tailored recommendations from other users.

  • Login: michael@test.com Password: password
  • Backend RESTful API using MVC architecture built with Ruby on Rails.
  • Frontend single page application built with npm, webpack, Vue.js, Bootstrap.
  • Secure authenticated user profiles using JSON web tokens.
  • Backend deployment with Fly.io and frontend deployment with Netlify.
  • Integrates Twilio messaging to allow users to send text notifications for recommendations.
  • Integrates Cloudinary file uploads to allow users to share photos.
